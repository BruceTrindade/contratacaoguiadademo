## 🚀 Como fazer o deploy:

### Opção 1: GitHub + Vercel (Recomendado)
1. Faça push para o GitHub
2. Conecte na Vercel
3. Deploy automático

### Opção 2: CLI da Vercel
```bash
npx vercel --prod
```

### Opção 3: Dashboard da Vercel
1. Acesse vercel.com
2. Importe o repositório
3. Deploy automático
